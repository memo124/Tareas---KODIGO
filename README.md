## Descripcion

Actividad formativa sobre la creacion de clases para la gestion de datos usando Programacion Orientada a Objetos en Python.

## Objetivo

Construir un pipeline de datos aplicando abstraccion, herencia, encapsulamiento, mixins y polimorfismo.

## Fases

1. Arquitectura base y mixins
2. Ingesta y transformacion de datos
3. Analisis y orquestacion del pipeline

## Componentes principales

- `DataComponent`: clase abstracta base.
- `ValidatorMixin`: validacion de archivos y columnas.
- `LoggerMixin`: seguimiento de mensajes en consola.
- `DataLoader`: carga de datos desde CSV.
- `DataCleaner`: limpieza y transformacion de datos.
- `DataAnalyst`: analisis y generacion de reportes.
